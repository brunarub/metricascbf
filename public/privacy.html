Política de Privacidade

Este dashboard coleta e exibe métricas das contas oficiais da CBF nas redes sociais (Instagram, YouTube, TikTok) para uso exclusivo da equipe interna de marketing.

Dados coletados

Acessamos apenas dados públicos e métricas das contas autorizadas: visualizações, curtidas, comentários, compartilhamentos e alcance dos conteúdos publicados.

Uso dos dados

Os dados são utilizados exclusivamente para análise de performance de conteúdo pela equipe interna. Não são compartilhados com terceiros.

Armazenamento

Os tokens de acesso às APIs são armazenados de forma segura no servidor e utilizados apenas para leitura de métricas.
